# TickIt

TickIt helps you plan and manage your daily tasks, including a dedicated timeline view to visualize your day. Organize tasks by workspace categories such as Personal, Work, Freelance, and Projects. Add subtasks, set color-based priorities, and stay focused with an integrated Pomodoro timer. Powered by the `Hive` database for fast and secure local storage.
